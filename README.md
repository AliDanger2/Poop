<p align="center"><strong><span style="color:#ffaa00;">FEATURES</span></strong></p>

- <span style="color:#00b300;">/poop</span> - Toggle poop mode and crouch to poop!  
- <span style="color:#00b300;">/diarrhea</span> - Activate *explosive diarrhea mode*!  
- <span style="color:#00b300;">/toppoop</span> - Shows the poop leaderboard  
- <span style="color:#00b300;">/plunge</span> - Player lunge ability with flying poop trail!  
- <span style="color:#00b300;">/pa</span> - Admin control for plugin settings  
- Lightweight, no lag, and works perfectly with 1.20‚Äì1.21+
